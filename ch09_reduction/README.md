## 📝 核心概念

### 并行归约思想

```
输入: [1, 2, 3, 4, 5, 6, 7, 8]

第1步: [1+2, 3+4, 5+6, 7+8] = [3, 7, 11, 15]
第2步: [3+7, 11+15] = [10, 26]
第3步: [10+26] = [36]

输出: 36
```

### 优化版本演进

| 版本 | 优化点 | 问题 |
|------|--------|------|
| V1 | 基础实现 | Warp Divergence |
| V2 | 消除 Divergence | Bank Conflict |
| V3 | 顺序寻址 | 空闲线程多 |
| V4 | 首次加载时加法 | 仍有优化空间 |
| V5 | Warp 展开 | - |
| V6 | 完全展开 | 最优 |
